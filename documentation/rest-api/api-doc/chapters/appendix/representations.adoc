== Basic Types

[[timestamp-representation]]
=== Timestamps

All timestamps used in this API are in https://wikipedia.org/wiki/ISO_8601[ISO 8601] format.
Although nanosecond precision is available currently, clients MUST NOT rely on it, as it might be subject to change.

[[error-represenation]]
== Errors

This section lists all domain errors that can be thrown by the REST-API.
Responses follow the https://datatracker.ietf.org/doc/html/rfc9457[RFC 9457] Problem Details for HTTP APIs format.
The fields `type`, `status`, `title` and `instance` are part of _every_ problem response.
