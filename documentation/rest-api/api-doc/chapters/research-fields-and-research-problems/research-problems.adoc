= Research Problems

Research problems are meta objects that help better classify the content within the ORKG.
The research problem is essentially a <<Resources,resource>> and hence inherits all the specifications of resources.

== Listing Research Problems

include::{snippets}/research-problems_find-all/description.adoc[]

This endpoint supports the following additional sorting parameters:

|===
|Name |Description

|`research_problem_count`
|Sorts the returned research problems by the count of associated research fields.
A research problem is considered associated when there exists a path from a research problem to a research field in the form of: (Problem)<-[P32]-(Contribution)<-[P31]-(Paper)-[P30]->(ResearchField)

|===
